Hugin
Open Source | C++, Linux Network Scanner

Hugin is a high-performance, multithreaded port scanner written in C++. It is built for speed, efficiency, and flexibility, making it a powerful tool for network reconnaissance. Hugin can scan all 65535 ports via TCP Connect (complete handshake) in 1 minute.
